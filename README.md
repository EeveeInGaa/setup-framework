# Rust Setup Framework

A small Rust Learning project to simplify and sped up the setup process. 
Currently, it only sets up React, a setup for Angular is planned. 
There should be both a simple (one click, most common setup) and an advanced (select every aspect) setup option.

## Current Features

- Simple Setup (one click); Advanced Setup planned
- React 19
- Tailwind CSS v4 (including design system)
- Biome (formatting + linting)
- React Router (with basic routes)
- predefined folder structure
- Path aliases
- basic global styles (including darkmode)
- responsive basic layout 
- predefined basic components
- cleanup/adjust vite default files
- install dependencies
- initiate git with main branch and initial commit
- select interface in terminal with details
